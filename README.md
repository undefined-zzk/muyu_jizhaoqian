# 木鱼福签 🙏

一款佛系治愈的微信小程序，专注于解压祈福、静心修行。基于 React 参考设计重新实现的微信小程序原生版本。

## ✨ 核心功能

### 1. 电子木鱼敲击
- 指尖轻触木鱼，播放音效反馈
- 功德值累积系统，记录每次敲击
- 木鱼晃动动画效果
- 功德 +1 浮动文字提示
- 木鱼棒敲击动画
- 自动敲击功能（每秒自动敲击一次）
- 长按 3 秒触发彩蛋特效（祥云/星星飘落 + 随机功德奖励）

### 2. 每日福签抽签
- 每日限抽 1 次专属福签
- 8 种吉祥签文，全吉无凶
- 精美签文卡片展示（大吉、上上签、吉、上吉）
- 抽签动画效果（旋转展开）
- 支持保存签文图片到相册
- 支持分享好运到朋友圈
- 已抽签可随时查看当日签文

### 3. 功德等级系统
- 10 个功德等级（初心者 → 功德圆满）
- 实时显示当前等级和进度
- 等级说明弹窗，查看所有等级详情
- 达到功德圆满（1000000）触发特殊效果

### 4. 祈愿墙
- 输入心愿生成电子祈愿牌
- 心愿保存到本地祈福墙
- 50 字心愿限制，最多 100 条
- 心愿列表展示，记录日期
- 心愿数量角标提示
- 可拖拽的祈愿墙入口按钮

### 5. 极简背景切换
- 6 款精美主题背景：
  - 禅意渐变（琥珀色系）
  - 祥云（蓝紫粉渐变）
  - 静心（绿青色系）
  - 暖阳（黄橙色系）
  - 素白（纯色）
  - 米黄（纯色）
- 一键切换背景皮肤
- 可拖拽的背景切换按钮
- 渐变色系，护眼舒适

### 6. 功德分享
- 生成精美的功德统计图片
- 显示当前等级、今日功德、总功德
- 功德圆满时显示特殊祝福语录
- 支持分享到朋友圈

## 🎯 技术实现

### 核心技术
- 微信小程序原生开发
- 本地存储（wx.setStorageSync）
- CSS3 动画效果
- 音频播放 API（wx.createInnerAudioContext）
- Canvas 绘图（生成分享图片）
- 震动反馈 API（wx.vibrateShort）

### 动画效果
- 木鱼晃动：rotate + scale 组合动画
- 功德浮动：translateY + opacity 渐变
- 抽签展开：slideUp + rotateY 3D 旋转
- 彩蛋特效：多元素下落 + 中心弹窗
- 祈愿列表：slideIn 滑入动画

### 数据存储
- totalMerit：总功德值
- todayMerit：今日功德值
- lastMeritDate：最后敲击日期
- lastDrawDate：最后抽签日期
- currentFortune：当日签文
- wishes：祈愿列表
- bgIndex：背景主题索引
- meritLevel：当前功德等级

## 📱 功能详解

### 木鱼交互
- 普通点击：功德 +1，播放音效，晃动动画
- 长按 3 秒：触发彩蛋，随机功德奖励（6-168），祥云/星星特效
- 自动敲击：开启后每秒自动敲击一次
- 功德圆满：达到 1000000 功德后显示特殊效果

### 抽签机制
- 每日 0:00 自动重置抽签次数
- 随机从 8 种吉签中抽取
- 签文包含：标题、内容、祝福语、日期
- 抽签后自动保存，可随时查看
- 已抽签时点击按钮会提示并显示当日签文

### 功德等级
- 10 个等级：初心者、虔诚者、修行者、善信、居士、善士、大善人、菩萨心、罗汉果、功德圆满
- 等级范围：0 → 100 → 500 → 1000 → 5000 → 10000 → 50000 → 100000 → 500000 → 1000000
- 实时显示进度条和下一等级所需功德
- 升级时显示提示
- 点击"等级说明"查看所有等级详情

### 祈愿墙
- 支持添加多条心愿（最多 100 条）
- 心愿按时间倒序排列
- 显示心愿内容和日期
- 心愿数量角标提示
- 可删除心愿
- 祈愿墙入口可拖拽到屏幕任意位置

### 背景主题
- 6 款精心设计的背景
- 渐变色和纯色两种风格
- 切换后自动保存偏好
- 背景切换按钮可拖拽

### 功德分享
- 生成精美的功德统计图片
- 普通状态：显示等级、今日功德、总功德
- 功德圆满状态：显示庆祝图标、特殊祝福语录
- 支持保存到相册或分享到朋友圈

## 🎁 彩蛋功能

长按木鱼 3 秒，触发隐藏祝福：
- 100 个祥云/星星/彩带从天而降
- 中心显示随机祝福语（福气满满、好运连连、心想事成等）
- 额外获得随机功德值（6、8、10、18、28、66、88、168）
- 震动反馈增强仪式感
- 自动敲击时无法触发彩蛋

- 优化了动画性能
- 使用原生 API 替代 Web API

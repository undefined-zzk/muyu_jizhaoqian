# 木鱼吉兆签 🙏

一款佛系治愈的微信小程序，专注于解压祈福、静心修行。基于 React 参考设计重新实现的微信小程序原生版本。

## ✨ 核心功能

### 1. 电子木鱼敲击
- 指尖轻触木鱼，实时震动反馈
- 功德值累积系统，记录每次敲击
- 木鱼晃动动画效果
- 功德 +1 浮动文字提示
- 长按 3 秒触发彩蛋特效（祥云/星星飘落 + 功德 +10）

### 2. 每日吉兆抽签
- 每日限抽 1 次专属吉签
- 8 种吉祥签文，全吉无凶
- 精美签文卡片展示（大吉、上上签、吉、上吉）
- 抽签动画效果（旋转展开）
- 支持保存签文图片
- 支持分享好运到朋友圈

### 3. 祈愿墙
- 输入心愿生成电子祈愿牌
- 心愿保存到本地祈福墙
- 100 字心愿限制
- 心愿列表展示，记录日期
- 心愿数量角标提示

### 4. 极简背景切换
- 6 款精美主题背景：
  - 禅意渐变（琥珀色系）
  - 祥云（蓝紫粉渐变）
  - 静心（绿青色系）
  - 暖阳（黄橙色系）
  - 素白（纯色）
  - 米黄（纯色）
- 一键切换背景皮肤
- 渐变色系，护眼舒适

## 🎨 设计特色

- 佛系治愈风格，极简轻量
- 全吉无凶，纯正向心理暗示
- 零门槛操作，打开即用
- 禅意 UI 设计，无广告干扰
- 流畅的动画效果
- 震动反馈增强真实感

## 🎯 技术实现

### 核心技术
- 微信小程序原生开发
- 本地存储（wx.setStorageSync）
- CSS3 动画效果
- 震动反馈 API（wx.vibrateShort）

### 动画效果
- 木鱼晃动：rotate + scale 组合动画
- 功德浮动：translateY + opacity 渐变
- 抽签展开：slideUp + rotateY 3D 旋转
- 彩蛋特效：多元素下落 + 中心弹窗
- 祈愿列表：slideIn 滑入动画

### 数据存储
- meritCount：功德值累计
- lastDrawDate：最后抽签日期
- currentFortune：当日签文
- wishes：祈愿列表
- bgIndex：背景主题索引

## 📱 功能详解

### 木鱼交互
- 普通点击：功德 +1，震动反馈，晃动动画
- 长按 3 秒：触发彩蛋，功德 +10，祥云/星星特效

### 抽签机制
- 每日 0:00 自动重置抽签次数
- 随机从 8 种吉签中抽取
- 签文包含：标题、内容、祝福语、日期
- 抽签后自动保存，可随时查看

### 祈愿墙
- 支持添加多条心愿
- 心愿按时间倒序排列
- 显示心愿内容和日期
- 心愿数量角标提示

### 背景主题
- 6 款精心设计的背景
- 渐变色和纯色两种风格
- 切换后自动保存偏好

## 🎁 彩蛋功能

长按木鱼 3 秒，触发隐藏祝福：
- 20 个祥云/星星从天而降
- 中心显示"福气满满"弹窗
- 额外获得 10 点功德值
- 震动反馈增强仪式感

## 📝 参考设计

本项目基于 React + Framer Motion 的 Web 设计重新实现为微信小程序版本：
- 保留了所有核心功能
- 适配微信小程序交互规范
- 优化了动画性能
- 使用原生 API 替代 Web API

参考设计位于 `木鱼吉兆签小程序设计/` 目录。
